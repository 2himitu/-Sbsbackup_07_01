# 게시판 사이트

작성, 수정, 삭제, 좋아요기능은 로그인시 이용 가능하다.
조회시 조회수가 증가한다.
수정, 삭제는 작성만이 가능하다.
아이디,비밀번호 찾기가 가능하다.
비밀번호 찾기시 가입시 입력한 메일에 임시 비밀번호를 전송한다.

관리자의 계정으로는 유저리스트를 통해 일반유저의 계정 삭제가 가능하다.
